What is JSX, and why is it used?
JSX is mainly stand for Javascript XML. It is syntax extension which contain HTML like code inside Javascript.

What is the difference between State and Props?
Both State and Props are used for managing and passing data in components. 
Props are read only and pass from parent to child.
State can modify inside a component and managed by component itself.

What is the useState hook, and how does it work?
It is a built-in React hook that allows to store and manage dynamic data. 

It is works by storing a slice of state inside a React component and providing a function to update the state. React automatically re-render the component to display the new value.

How can you share state between components in React?
State can be shared between components with lifting the state up to a common parent component.

How is event handling done in React?
By passing a function as the event handler.
